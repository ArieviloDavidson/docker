# Docker Playground

```bash
docker run hello-world
```

Teste esse comando com docker instalado na sua máquina para começar a se aventurar no mundo docker.

* helloWorldPython
Projeto simples com qualquer código em python, fique a vontade para brincar com APIs, servidores, sites, etc.

```bash
docker build -t app-python .
```

```bash
docker run app-python
```
